# 🏃 Athlete Tracker App

A simple Kotlin console application that manages **Athletes** and **Brands**, allowing CRUD operations and relationships between the two. Built with clean separation of concerns using models, controllers, and a main UI loop.

---

## ✨ Features

- ✅ Add, edit, delete Athletes and Brands
- ✅ Assign Athletes to Brands
- ✅ List Athletes by Brand
- ✅ View brand summary with athlete counts
- ✅ Prevent duplicate IDs
- ✅ Data validation using `readLine()?.toIntOrNull()`
- ✅ Modular design using APIs (controllers) and models
- ✅ Uses lambdas for filtering and mapping
- ✅ Console UI with a simple interactive menu

---

## 🛠 Technologies

- Kotlin
- JUnit 5 (for testing)
